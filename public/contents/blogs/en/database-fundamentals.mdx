Databases can come in different types based on different needs. Here are the most common database types:

### 1. Relational Databases (SQL) ğŸ“Š

This is the most commonly used database type. Data is stored in tables, and relationships are established between tables.

```sql
-- Example customer table
CREATE TABLE customers (
    id INT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    email VARCHAR(100),
    phone VARCHAR(15)
);

-- Example order table
CREATE TABLE orders (
    order_id INT PRIMARY KEY,
    customer_id INT,
    product_name VARCHAR(100),
    price DECIMAL(10,2),
    date DATE,
    FOREIGN KEY (customer_id) REFERENCES customers(id)
);
```

Features:
- âœ… Data is stored in organized tables
- âœ… Relationships can be established between tables (e.g., customer-order relationship)
- âœ… Supports ACID principles (Atomicity, Consistency, Isolation, Durability)
- âœ… Complex queries can be performed

Popular Relational Databases:
