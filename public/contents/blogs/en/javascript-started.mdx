# Introduction to JavaScript: Basic Concepts and Examples 🚀

Hello! Welcome to your JavaScript learning journey. In this guide, we'll learn the basic concepts of JavaScript with a friendly approach and plenty of examples. Let's begin! 👋

## Hello World! 👋

As with every programming language, let's start learning JavaScript with the classic "Hello World" example:

```javascript
console.log("Hello World!");
alert("Hello World!");
```

## Code Structure 📝

JavaScript code runs line by line from top to bottom. Each statement can be separated by a semicolon (;):

```javascript
let message = "Hello";
console.log(message);

// You can write multiple statements in one line
let name = "John"; let age = 25;
```

## Modern Mode: "use strict" 🎯

In modern JavaScript, we use the "use strict" directive for safer code writing:

```javascript
"use strict";

// Now you're in modern mode with stricter rules
let test = 123; // ✅ Correct
test = 456;     // ❌ Error: variable not defined
```

## Variables 📦

Variables are like "boxes" we use to store data:

```javascript
// Declaring variables with let
let name = "Alice";
let age = 30;
let isStudent = true;

// Declaring constants with const
const PI = 3.14;
const BIRTH_YEAR = 1990;

// Declaring variables with var (old method)
var oldVariable = "old method";
```

## Data Types 🏷️

Basic data types in JavaScript:

```javascript
// Number
let number = 42;
let decimal = 3.14;

// String
let text = "Hello";
let name = 'John';

// Boolean
let isTrue = true;
let isFalse = false;

// Null and Undefined
let empty = null;
let undefined;

// Object
let person = {
    name: "Michael",
    age: 25
};

// Array
let colors = ["red", "blue", "green"];
```

## Type Conversions 🔄

You can convert one data type to another:

```javascript
// Converting to String
let value = 123;
let textValue = String(value); // "123"

// Converting to Number
let textNumber = "456";
let number = Number(textNumber); // 456

// Converting to Boolean
let numberBoolean = Boolean(1);     // true
let textBoolean = Boolean("");   // false
```

## Operators ➕

Basic mathematical and assignment operators:

```javascript
// Arithmetic operators
let a = 5 + 3;  // Addition
let b = 10 - 4; // Subtraction
let c = 3 * 4;  // Multiplication
let d = 15 / 3; // Division
let e = 10 % 3; // Modulus (remainder)

// Assignment operators
let x = 5;
x += 3;  // x = x + 3
x -= 2;  // x = x - 2
x *= 4;  // x = x * 4
x /= 2;  // x = x / 2
```

## Comparisons 🤔

Operators used for comparing values:

```javascript
let a = 5;
let b = 10;

console.log(a < b);   // true
console.log(a > b);   // false
console.log(a <= 5);  // true
console.log(b >= 10); // true
console.log(a === 5); // true (checks value and type)
console.log(a !== b); // true
```

## Interaction: Alert, Prompt, and Confirm 💬

Methods used for interacting with users:

```javascript
// Displaying information
alert("Hello! This is an alert message.");

// Getting user input
let name = prompt("What's your name?", "Guest");
console.log("Hello, " + name);

// Getting confirmation
let confirmation = confirm("Do you want to continue?");
if (confirmation) {
    console.log("Action confirmed!");
} else {
    console.log("Action cancelled!");
}
```

## Conditional Operators: if and ? 🔍

Structures used for conditional operations:

```javascript
// if-else structure
let age = 18;

if (age >= 18) {
    console.log("You are an adult!");
} else {
    console.log("You are not an adult!");
}

// Ternary operator (?)
let status = age >= 18 ? "Adult" : "Not adult";
```

## Logical Operators 🧮

Operators used for combining conditions:

```javascript
let age = 25;
let hasLicense = true;

// AND (&&)
if (age >= 18 && hasLicense) {
    console.log("You can drive!");
}

// OR (||)
let isHoliday = true;
let isWeekend = false;

if (isHoliday || isWeekend) {
    console.log("You can rest today!");
}

// NOT (!)
let isWorking = false;
console.log(!isWorking); // true
```

## Loops: while and for 🔄

Structures used for running code repeatedly:

```javascript
// while loop
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// for loop
for (let j = 0; j < 5; j++) {
    console.log(j);
}

// Loop over array
let fruits = ["apple", "pear", "banana"];
for (let fruit of fruits) {
    console.log(fruit);
}
```

## Switch Statements 🔀

Structure used for multiple condition cases:

```javascript
let day = "Monday";

switch (day) {
    case "Monday":
        console.log("First day of the week");
        break;
    case "Friday":
        console.log("Weekend is coming");
        break;
    case "Saturday":
    case "Sunday":
        console.log("It's weekend!");
        break;
    default:
        console.log("Regular day");
}
```

## Functions 🛠️

Structures used for making code reusable:

```javascript
// Simple function
function greet(name) {
    return "Hello, " + name + "!";
}

console.log(greet("Alice")); // "Hello, Alice!"

// Function with parameters
function add(a, b) {
    return a + b;
}

console.log(add(5, 3)); // 8
```

## Function Expressions 📝

You can assign functions to variables:

```javascript
// Function expression
let greet = function(name) {
    return "Hello, " + name + "!";
};

console.log(greet("Michael")); // "Hello, Michael!"
```

## Arrow Functions ➡️

Short function syntax used in modern JavaScript:

```javascript
// Single-line arrow function
let square = x => x * x;
console.log(square(4)); // 16

// Multi-line arrow function
let add = (a, b) => {
    let result = a + b;
    return result;
};

console.log(add(5, 3)); // 8

// Arrow function without parameters
let greet = () => "Hello!";
console.log(greet()); // "Hello!"
```
