## Kod Kalitesi: Temiz ve Bakımı Kolay Kod Yazma 📝

Kod kalitesi, yazılımın okunabilirliğini ve bakımını kolaylaştıran en önemli faktörlerden biridir. İşte bazı temel prensipler:

### 1. DRY (Don't Repeat Yourself) Prensibi

```javascript
// ❌ Kötü örnek
function selamlaAhmet() {
    console.log("Merhaba Ahmet! Nasılsın?");
}

function selamlaMehmet() {
    console.log("Merhaba Mehmet! Nasılsın?");
}

// ✅ İyi örnek
function selamla(isim) {
    console.log(`Merhaba ${isim}! Nasılsın?`);
}

selamla("Ahmet");
selamla("Mehmet");
```

### 2. KISS (Keep It Simple, Stupid)

```javascript
// ❌ Karmaşık örnek
function yasHesapla(dogumTarihi) {
    const bugun = new Date();
    const dogum = new Date(dogumTarihi);
    const fark = bugun - dogum;
    const yasMillisaniye = new Date(fark);
    return Math.abs(yasMillisaniye.getUTCFullYear() - 1970);
}

// ✅ Basit örnek
function yasHesapla(dogumYili) {
    const buYil = new Date().getFullYear();
    return buYil - dogumYili;
}
```

### 3. Anlamlı İsimlendirme

```javascript
// ❌ Kötü isimlendirme
let x = 5;

// ✅ İyi isimlendirme
let yasSiniri = 5;
let sayilar = [1, 2, 3];

```

### 2. Breakpoint Kullanımı

```javascript
function hesaplaToplamFiyat(urunler) {
    let toplam = 0;
    
    // Buraya breakpoint ekleyebilirsiniz
    for(let urun of urunler) {
        toplam += urun.fiyat;
    }
    
    return toplam;
}

// Kod içinde debugger kullanımı
function kontrolEt(deger) {
    debugger; // Tarayıcı burada duracak
    if (deger < 0) {
        throw new Error("Negatif değer!");
    }
    return deger * 2;
}
```

## Polyfills: Eski Tarayıcı Desteği 🔄

Polyfill'ler, modern JavaScript özelliklerini eski tarayıcılarda kullanmamızı sağlar:

### 1. Array.includes() için Polyfill

```javascript
if (!Array.prototype.includes) {
    Array.prototype.includes = function(aranacak) {
        return this.indexOf(aranacak) !== -1;
    }
}

// Artık eski tarayıcılarda da kullanabilirsiniz
let meyveler = ["elma", "armut", "muz"];
console.log(meyveler.includes("elma")); // true
```

### 2. String.padStart() için Polyfill

```javascript
if (!String.prototype.padStart) {
    String.prototype.padStart = function(uzunluk, dolguKarakteri) {
        dolguKarakteri = dolguKarakteri || ' ';
        return dolguKarakteri.repeat(Math.max(0, uzunluk - this.length)) + this;
    }
}

// Kullanım
console.log("5".padStart(2, "0")); // "05"
```

## Kodlama Stili 🎨

Tutarlı bir kodlama stili, kodunuzu daha okunabilir ve bakımı kolay hale getirir:

### 1. Girintileme ve Boşluklar

```javascript
// ❌ Kötü girintileme
function hesapla(a,b){
if(a>b){return a+b}
    else{return a-b}
}

// ✅ İyi girintileme
function hesapla(a, b) {
    if (a > b) {
        return a + b;
    } else {
        return a - b;
    }
}
```

### 2. Değişken ve Fonksiyon İsimlendirme

```javascript
// Değişkenler için camelCase
let ogrenciAdi = "Ahmet";
let maksimumPuan = 100;

// Sabitler için UPPER_SNAKE_CASE
const MAKSIMUM_DENEME_SAYISI = 3;
const API_URL = "https://api.ornek.com";

// Sınıflar için PascalCase
class OgrenciYonetimi {
    constructor() {
        // ...
    }
}
```

### 3. Modern JavaScript Özellikleri

```javascript
// ❌ Eski stil
var eskiDegisken = "eski";
function topla(a, b) {
    return a + b;
}

// ✅ Modern stil
let degisken = "modern";
const sabit = "değişmez";

// Ok fonksiyonları
const topla = (a, b) => a + b;

// Template literals
const mesaj = `Merhaba ${isim}!`;

// Destructuring
const { ad, yas } = kullanici;
const [ilk, ikinci] = dizi;
```
